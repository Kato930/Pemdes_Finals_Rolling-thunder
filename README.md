Instruction Manual:

First make a folder called Pemdes Finals and add al the files in this git repo in that folder.

Second add the .sql database in xampp php admin.

Third run cmd terminal.

**Terminal 1 (Video AI):** Run `python test.py` in the `Video Model` folder.

**Terminal 2 (Frontend):** Run `npm run dev` in the `SALA` folder.

**Terminal 3 (Mock Hardware):** Run `python mock_arduino.py` in the Pemdes Finals folder.
